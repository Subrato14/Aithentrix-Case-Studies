#  EdTech Learning Management Platform

A modern LMS (Learning Management System) designed for educational institutes, coaching centers, universities, and independent educators. Includes online courses, video streaming, quizzes, assignments, certificates, and live classes.

---

##  Project Overview
This platform allows students to learn online at their own pace while instructors create and manage courses with deep analytics.

The project contains:

- **Student Web Panel**
- **Instructor Web Panel**
- **Admin Web Panel**

---

##  Tech Stack
**Frontend:**  
- React.js  
- Next.js (Admin Dashboard)  
- Tailwind / Material UI  

**Backend:**  
- Node.js (Express)  
- JWT Authentication  

**Database:**  
- MongoDB (Mongoose ORM)

**Cloud & Storage:**  
- AWS S3  
- AWS CloudFront  

**Payments:**  
- Stripe / Razorpay
