#  E-Commerce Website for Fashion Brand 
**Industry:** Online Retail   
**Duration:** 16.5 weeks  
**Role:** Aithentrix (CEO — Project Manager & Delivery Oversight)

##  Overview
A mobile-responsive e-commerce website for a fashion retailer to increase online sales and automate ordering.

## Solution  
A full-featured e-commerce platform was built with:

- Customer-facing storefront  
- Product catalogue & advanced filters  
- Secure checkout with payment gateway  
- Admin dashboard for inventory & orders  
- Coupon engine + user account system  

---

## Key Features Delivered  
- Product listings with categories  
- Cart & checkout flow  
- Order tracking  
- Integrated online payments  
- Admin CMS for products & banners  
- Inventory & low-stock alerts  
- Customer reviews & ratings  

---

## Tech Stack (Recommendation)  
**Frontend:** React  
**Backend:** Node.js + Express  
**Database:** MongoDB  
**Payments:** Razorpay / Stripe  
**Hosting:** AWS (EC2 + CloudFront + S3)

## Backend
1. Go to `/backend`
2. Install dependencies → `npm install`
3. Start server → `npm start`
4. API Endpoints:
   - GET /api/products
   - GET /api/categories
   - POST /api/cart (add item)

## Frontend
1. Go to `/frontend`
2. Install dependencies → `npm install`
3. Start UI → `npm start`
4. Backend must be running on port 5002.
