#  E-Commerce Website for Fashion Brand

##  Overview
A mobile-responsive e-commerce website for a fashion retailer to increase online sales and automate ordering.

##  Duration
3 months

##  Team Size
4 (Frontend Dev, Backend Dev, UI/UX Designer, QA)

##  Tech Stack
React • Node.js • Express • MongoDB • Razorpay

##  Scope of Work
- Full website development
- Admin panel creation
- Category & product management
- Checkout & payment gateway integration

##  Key Features
- Product filters  
- Cart & checkout  
- Order history  
- Admin dashboard  

##  Deliverables
- Source code  
- UI/UX wireframes  
- API documentation  
- Admin training  

##  My Role (CEO)
- Defined requirements  
- Coordinated with development team  
- Ensured timely delivery  
- Managed client communication  

##  Impact
- Boosted online revenue by ~3×

## Backend
1. Go to `/backend`
2. Install dependencies → `npm install`
3. Start server → `npm start`
4. API Endpoints:
   - GET /api/products
   - GET /api/categories
   - POST /api/cart (add item)

## Frontend
1. Go to `/frontend`
2. Install dependencies → `npm install`
3. Start UI → `npm start`
4. Backend must be running on port 5002.
